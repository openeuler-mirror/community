# openEuler ROS SIG

- 在openEuler社区中添加对ROS和ROS2的支持
- 根据openEuler迭代版本，持续完成ROS和ROS2中各个组件向openEuler的移植，并提供ROS和ROS2在openEuler上的使用文档
- 及时响应用户反馈，解决相关问题
