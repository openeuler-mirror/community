# QT SIG

## SIG组职责
- 根据openEuler迭代版本，持续完成qt6中各个组件向openEuler的移植
- 负责 qt 相关软件包的规划、维护和升级
- 及时响应用户反馈，解决相关问题


# 组织会议

- 公开的会议时间：北京时间，每周三 下午，4点~5点
