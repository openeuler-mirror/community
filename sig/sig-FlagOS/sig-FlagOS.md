# openEuler FlagOS Special Interest Group (SIG)

English | [简体中文](./sig-FlagOS_cn.md)

The openEuler FlagOS SIG aims at migrating FlagOS to openEuler, in order to reduce the threshold for using openEuler and promote openEuler to new users from FlagOS Community.


## SIG Mission and Scope

### Mission

- Add FlagOS support for openEuler community.
- According to the iterative version of openEuler, constantly complete the migration of FlagOS to openEuler, and provide scripts for building FlagOS components on openEuler and documents.
- Respond to users' feedback in a timely manner and resolve the problems.

### Scope

- Migrate FlagOS components to openEuler, with further maintenance and upgrades
- Create and update FlagOS binary tarballs for openEuler, provide build scripts and documents
- Manage the documentations, meetings, mailing list and IRC within FlagOS SIG

### Deliverables

- Source and tarballs.

### Repositories and description managed by this SIG

Repository of scripts for building FlagOS for openEuler and documents: https://atomgit.com/openeuler/FlagOS
