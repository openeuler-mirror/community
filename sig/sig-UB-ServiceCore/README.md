# SIG名称
sig-UB-ServiceCore作为OS高阶服务，负责支持UB灵衢超节点系统高阶服务演进与维护，主要目标：

- 负责UB ServiceCore组件架构看护、新特性设计评审、代码审核，并协同sig-UnifiedBus、sig-Long共同探索超节点协同架构。

- 负责UB ServiceCore 开发、测试、漏洞修复、发布。

- 持续开放UB ServiceCore对外API运作机制，协同社区共建API规范

- 通过共建Tutorial、API文档、设计文档等关键技术规范，将灵衢相关技术赋能给开发者
