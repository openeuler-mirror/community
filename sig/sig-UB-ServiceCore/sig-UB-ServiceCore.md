## SIG Mission and Scope

### sig-UB-ServiceCore objectives:
- Responsible for the evolution and maintenance of the UB ServiceCore in the community.

### sig-UB-ServiceCore positioning:
- Responsible for overseeing the architecture of the UB ServiceCore component, reviewing new feature designs, conducting code reviews, and collaborating with sig-UnifiedBus and sig-Long to explore the super-node collaboration architecture.
- Responsible for the development, testing, vulnerability fixing, and release of UB ServiceCore.
- Continuously build the operational mechanisms of UB ServiceCore's external APIs and work with the community to jointly establish API standards.
- Empower developers with UB-related technologies by co-creating key technical specifications such as tutorials, API documentation, and design documents.
