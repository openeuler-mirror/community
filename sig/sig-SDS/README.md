
# sig-SDS 愿景

愿景：汇聚存储英才，共同打造高质量、高性能、高可靠性的分布式存储组件版本，构建丰富的南北向生态

# 目标

- 紧跟原生社区，坚持“upstream first”原则，丰富ARM/openEuler分布式存储组件生态圈
- 针对ARM架构开展优化，打造极致性能
- 为openEuler发布版本的组件，决策版本、打造质量防护网，实现高质量发布
