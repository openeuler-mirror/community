# memsafety

​	随着科技发展，安全不再是当今世界的可选项。频发的安全漏洞，越来越严重的影响着每个人的工作和生活。Rust 语言天生为并发和安全而设计，并且借鉴了面向过程/面向对象/函数式等语言的特点，本sig主要涉及内存安全相关项目。
