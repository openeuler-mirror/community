# openEuler Edge SIG

## 目标

- 以OpenEuler为操作系统底座，面向边缘计算领域提供对应的中间件和解决方案
- 支持Intel，ARM,RISC-V等不同芯片平台
- 建立一个开放社区，提供相关技术交流，信息分享的地方

## 未来规划

- 将成熟的Edge相关组件移植到OpenEuler系统
- 探索一些新的场景，依托OpenEuler的优势，拓展Edge的领域
- 组织一些Meetup活动，交流信息，分享方案
