# Networking  SIG

- 在openEuler社区中添加对Networking的支持
- 负责Networking相关软件包的规划、选型和维护
- 负责将openEuler社区中的Networking的工作成果回馈给上游社区
- 及时响应用户反馈，解决相关问题
