# openEuler 鹏腾(鲲鹏、飞腾) & Arm SIG

## 愿景
助力鹏腾、Arm架构生态落地实施，促进鹏腾、Arm架构在通用计算产业的发展

## 业务范围
   - 加快鹏腾、Arm架构共性问题在上游社区的问题解决
   - 收集鹏腾、Arm架构用户生态需求，加强对社区Kernel SIG的支撑
   - 相关迁移工具、优化工具的联合分享以及合作开发演进
